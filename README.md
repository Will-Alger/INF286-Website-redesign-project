# INF286-Website-redesign-project
### http://www.irishwrecksonline.net will be redesigned for educational purposes

The objective of this site is to provide a catalog of shipwrecks around Ireland and provide information about them.




### Goal of project
##### *This project will be redesigning 8 pages*
* Update the asthetic and style.
* Improve website navigation.
* Improve presentation and organization of information.
